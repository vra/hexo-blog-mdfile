在Linux和Mac下，执行`du -sh` 来统计目录大小时，默认不统计以点开头的文件或者目录，比如`.local`，因此得到不准确的统计结果。从[这里](https://superuser.com/a/633808)知道，要统计隐藏文件和目录，可以这么用:
```bash
# 只统计隐藏文件或者目录
du -sh .[^.]*

# 统计隐藏文件或者目录和常规文件
du -sh .[^.]* *
```
结合 `sort -h`，可以快速得到比较大的目录：
```bash
du -sh .[^.]* * |sort -h
```
